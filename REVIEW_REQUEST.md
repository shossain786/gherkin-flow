## Template B — Community post (BDD Slack / Discord / mailing list)

**Subject:** Anyone tried GherkinFlow? Looking for honest feedback

Hey everyone,

I've been building [GherkinFlow](https://marketplace.visualstudio.com/items?itemName=RazaTech.gherkin-flow) — a VS Code extension that adds one-click Cucumber test running, step-level results, inline failure decorations, AI scenario generation, and a BDD quality linter.

It supports Java (Maven/Gradle), TypeScript/JavaScript (cucumber-js), and Python (Behave) with zero config for JS/Python.

If you've tried it (or are willing to give it a quick spin), I'd love a review on the Marketplace — even a short one describing your stack and what worked or didn't. Real-world feedback shapes the roadmap directly.

Link: https://marketplace.visualstudio.com/items?itemName=RazaTech.gherkin-flow&ssr=false#review-details

Happy to help with any setup issues — drop questions here.
